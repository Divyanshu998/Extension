How to Load the PhishGuard AI Extension

Step 1: Open Chrome Extensions Page

Open Google Chrome and navigate to chrome://extensions by typing it in the address bar.

Step 2: Enable Developer Mode

In the top-right corner of the Extensions page, toggle on "Developer mode".

Step 3: Load the Extension

Click the "Load unpacked" button that appears after enabling Developer mode.

Navigate to the PhishGuard AI extension folder (the folder containing the manifest.json file) and select it.

Step 4: Verify Installation

The PhishGuard AI extension should now appear in your list of extensions with its icon.

Make sure it's enabled (toggle switch is on).

Step 5: Test the Extension

Open the test pages to see the extension in action:

The extension should show different behaviors for each page:

  • For the legitimate page: Green "Safe" status
  • For the phishing page: Warning banner and red alert

Note: This is a prototype extension for demonstration purposes. In a real-world scenario, you would need to implement the actual AI models and cloud services.
